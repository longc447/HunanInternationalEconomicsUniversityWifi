# HunanInternationalEconomicsUniversityWifi
湖南涉外经济学院WiFi认证

# 使用说明
1.安装nodejs

2.安装git

3.克隆项目
```
git clone https://github.com/longc447/HunanInternationalEconomicsUniversityWifi.git
```

4.进入文件夹
```
cd HunanInternationalEconomicsUniversityWifi
```

5.安装依赖
```
npm install
```

6.运行
```
npm run dev
```
